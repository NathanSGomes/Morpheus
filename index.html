Ideias Iniciais

1ª App de venda

Minha ideia inicial era fazer um app de vendas de comida, mas percebi que muitos grupos tinham esse projeto, então resolvi partir para outra ideia, afinal fazer o mesmo que a maioria da um brecha comparativa gigantesta.

2ª Plat. de RPG

Pensei em fazer uma plataforma de RPG, mas quando fui ver os "problemas", a ausência de conhecimento da minha parte, como mexer com objetos 3D, fazer o jogador salvar a sala no próprio PC e etc... só não falo mais sobre porque ainda planejo fazer esse projeto rs.

3ª Banco de Dados

Nesse momento acho eu que viajei total, queria fazer um banco de dados com infográficos automáticos, sistema de avalição e média de clientes e contratantes, o real empecilho seria ligar tudo, perdi cerca de 2 semanas pensando como tudo funcionaria mas acabei abandonando essa ideia.

4ª Bot

Depois de passar 2 meses tive a ideia de fazer um bot, uma característica minha é ser preguiçoso, então porque não ir de encontro a automação, aproveitei o feriado do dia das crianças e fiquei cerca de 16hrs durante 5 dias pesquisando e criando meu "filho" sempre fui aficionado em livros de terror e me senti o próprio Victor.

HTML, CSS e JavaScript

A ideia inicial de Morpheus sempre foi servir como uma apresentação para Orpheus, queria criar uma página que tinham característica da minha personalidade e referências ao nome do mesmo, foi bem divertido todo processo criativo mesmo alguns amigos me xingando por usar CSS puro rs.

Linguagens Utilizadas

Python

Minha experiência cpm Python foi mais satisfatória do que imaginava, eu tenho TOC de acentuação e trabalhar com uma linguagem que os acentos não geram erros no terminal é sensacional, prentendo aprender cada vez mais sobre automação com Python para realizar cada menos tarefas.

HTML

No começo achei que seria mais trabalhoso mexer com HTML, percebi que é mais intuitivo que algumas linguagens de programação (e sim sei que HTML não é uma linguagens de programação), mas imaginei que quebraria mais a cabeça para enteder certas coisas.

CSS

O que falar de CSS? É um ótimo passatempo, brincar com efeitos, cores e etc... no geral acho muito legal chegar a 500 linhas só para ajustar detalhes mínimos, talvez seja só meu lado artístico pendendo pra um lado, porém passaria mais tempo testando cores, formatos e tamanhos.

JavaScript

Infelizmente não tive muito espaço para aprender sobre JavaScript, já que só o usei para escrever 2 linhas de código, daria para fazer um bot via JavaScript, mas sabe né Python > JavaScript (prioridades).
